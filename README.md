# SQL Practice & Analysis Portfolio

## Overview
This repository contains SQL exercises, and analyses covering a wide range of SQL concepts used in data analysis, reporting, and database management.

## SQL Topics Covered

### Database Querying
- SELECT statements
- Filtering with WHERE
- Sorting with ORDER BY
- Limiting results
- Aliasing columns and tables

### Joins
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL OUTER JOIN
- SELF JOIN

### Aggregations
- COUNT(), SUM(), AVG(), MIN(), MAX()
- GROUP BY
- HAVING

### Subqueries
- Scalar subqueries
- Correlated subqueries
- Nested queries

### Common Table Expressions (CTEs)
- Basic CTEs
- Multiple CTEs
- Recursive CTEs

### Window Functions
Worked with the three major categories of window functions:
- Aggregate Window Functions (SUM, AVG, COUNT, etc.)
- Ranking Window Functions (ROW_NUMBER, RANK, DENSE_RANK, NTILE)
- Analytical Window Functions (LAG, LEAD, FIRST_VALUE, LAST_VALUE)

### Case Statements
- Conditional logic using CASE WHEN
- Categorization and data transformation

### Data Cleaning
- Handling NULL values
- Removing duplicates
- String manipulation
- Date and time functions

### Advanced SQL
- Views
- Temporary Tables
- Set Operations (UNION, INTERSECT, EXCEPT)
- Stored Procedures (where applicable)

## Skills Demonstrated
- Data extraction and transformation
- Business reporting and KPI analysis
- Data cleaning and preparation
- Query optimization
- Analytical problem-solving

## Tools
- MySQL
- PostgreSQL
